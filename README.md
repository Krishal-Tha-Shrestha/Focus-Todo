# Focus — Task Tracker

A clean, minimal task tracker with a green aesthetic. Built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies, no build step.

**Live demo:** [krishal-tha-shrestha.github.io/Focus-Todo](https://krishal-tha-shrestha.github.io/Focus-Todo/todo-app.html)

---

## Features

**Task Management**
- Create topics (e.g. Study, Work, Personal)
- Add subtopics under each topic (e.g. Math, Physics, Chemistry)
- Add tasks with optional due dates inside each subtopic
- Mark tasks as complete with a checkbox
- Delete tasks, subtopics, or entire topics
- Filter tasks by All / Active / Done
- Progress bar showing completion per topic

**Calendar**
- Monthly calendar view with navigation
- Click any day to add tasks for that date
- Tag calendar tasks with a topic
- Days where all tasks are completed turn green automatically
- Color-coded dots show done vs pending tasks at a glance

**Search**
- Global search bar in the header
- Searches across all topics and subtopics instantly
- Check off or delete tasks directly from search results

**Dark Mode**
- Toggle between light and dark theme with one click
- Theme preference saved and remembered across sessions

**Persistence**
- All data stored in `localStorage` — no backend needed
- Works fully offline

## Usage

Just open `todo-app.html` in any browser. No installation required.

```bash
git clone https://github.com/Krishal-Tha-Shrestha/Focus-Todo.git
cd Focus-Todo
# open todo-app.html in your browser
```
